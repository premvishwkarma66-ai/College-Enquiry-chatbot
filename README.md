# College Enquiry Chatbot 🎓

A web-based chatbot built to help students get quick answers to college-related queries such as admissions, courses, fees, placements, and facilities.

## Features

* Instant student query response
* FAQ-based answer system
* Simple and interactive chat interface
* Stores enquiry logs
* Admin panel for managing data

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Flask
* **Database:** SQLite
* **Libraries:** NLTK, Scikit-learn

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/college-enquiry-chatbot.git
cd college-enquiry-chatbot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```

## Usage

Open the application in your browser and start asking college-related questions.
